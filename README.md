
# AI 일본 문화 체험 실습

관광일본어과 1반 학생을 위한 AI 교류수업용 인터랙티브 웹페이지입니다.

일본 신사의 대표적인 문화 요소인 참배 예절, 오미쿠지, 에마를 살펴보고, 원하는 테마를 선택하여 Gemini Canvas에 입력할 웹앱 제작 프롬프트를 만들 수 있습니다.

## 주요 기능

- 35분 수업 시간 배분 안내
- 일본 신사 문화 핵심 요소 소개
  - 二礼二拍手一礼
  - おみくじ
  - 絵馬
- 일본 신사 테마 선택
- Gemini Canvas용 프롬프트 자동 변경
- 전체 프롬프트 복사
- 디자인, 일본어, 오류 수정 프롬프트 제공

## 사용 방법

1. GitHub Pages 링크로 웹페이지에 접속합니다.
2. 원하는 일본 신사 테마를 선택합니다.
3. `전체 프롬프트 복사하기` 버튼을 누릅니다.
4. 복사한 내용을 Gemini Canvas에 붙여넣습니다.
5. 생성된 웹앱을 확인하고 수정 프롬프트를 활용해 개선합니다.

## 프로젝트 구성

```text
.
├── index.html
└── README.md
````

별도의 설치나 서버 실행은 필요하지 않습니다.

## 기술 구성

* HTML
* CSS
* JavaScript
* Tailwind CSS CDN
* Chart.js CDN

## 실행 방법

로컬에서는 `index.html` 파일을 브라우저로 열면 바로 실행됩니다.

## GitHub Pages 게시

1. GitHub 저장소에 `index.html`과 `README.md`를 업로드합니다.
2. 저장소의 `Settings`로 이동합니다.
3. `Pages` 메뉴를 선택합니다.
4. 배포 소스를 `Deploy from a branch`로 설정합니다.
5. 브랜치는 `main`, 폴더는 `/root`로 선택합니다.
6. 저장 후 생성된 GitHub Pages 주소를 공유합니다.

## 수업 목적

이 프로젝트는 코딩 문법 자체를 학습하기보다, 일본어와 일본 문화에 대한 전공 지식을 AI를 활용해 디지털 콘텐츠로 구체화하는 경험을 제공하는 데 목적이 있습니다.

학생은 AI가 생성한 일본어와 문화 설명을 그대로 받아들이지 않고, 전공 지식을 바탕으로 표현과 내용의 정확성을 확인하고 수정합니다.

## 참고

* 일본어 발음은 한글로 표기하지 않습니다.
* 한자 읽기가 필요한 경우 히라가나를 사용합니다.
* 한국어는 발음 안내가 아니라 의미와 문화 설명에 사용합니다.
* 생성되는 웹앱은 문화 체험용 콘텐츠입니다.
