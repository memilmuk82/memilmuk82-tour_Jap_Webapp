신사의 3대 핵심 콘텐츠

앱에 반드시 들어가야 할 일본 전통 문화 요소입니다.

🙇‍♂️

二礼二拍手一礼

두 번 절하고, 두 번 박수 치고, 한 번 절하는 참배 예절입니다. 버튼 클릭 시 순차적으로 안내되도록 구성합니다.

🏮

おみくじ

통을 흔들어 오늘의 운세를 뽑는 체험입니다. 대길(大吉)부터 소길(小吉)까지 무작위 결과를 제공합니다.

🪵

絵馬

나무판에 소원을 적어 벽에 거는 행위입니다. 이름과 소원 메시지를 입력받아 리스트에 추가하는 기능을 넣습니다.

STEP 01 테마 선택 및 프롬프트 생성

원하는 테마를 클릭하면 하단의 프롬프트가 즉시 변경됩니다.

Prompt Preview
나는 관광일본어과 학생이야. 일본 신사 문화를 간단하게 체험할 수 있는 모바일 웹앱을 만들어줘. 전체 분위기는 [벚꽃이 흩날리는 교토의 봄 신사] 테마로 만들어줘. 다음 기능을 넣어줘. 1. 참배 예절 체험 - 일반적인 신사 참배 방식인 二礼二拍手一礼을 안내해줘. - 일본어 표기 아래에 히라가나 읽기인 にれい・にはくしゅ・いちれい를 표시해줘. - 절하기, 박수 두 번, 마지막 절하기 버튼을 순서대로 누르면 “参拝を終えました。”라는 완료 문구와 한국어 의미를 보여줘. 2. 오미쿠지 운세 뽑기 - “おみくじを 引く” 버튼을 누르면 카드가 잠시 흔들리게 해줘. - 大吉、吉、中吉、小吉 중 하나가 무작위로 나오게 해줘. - 각 결과에는 일본어 표기, 히라가나 읽기, 한국어 의미, 짧은 일본어 행운 메시지를 보여줘. 3. 에마 소원 걸기 - 이름과 소원을 입력할 수 있게 해줘. - “絵馬를 掛ける” 버튼을 누르면 아래 소원판에 에마 카드가 추가되게 해줘. 언어 표기 원칙: - 일본어 표기, 히라가나 읽기, 한국어 의미를 시각적으로 명확히 구분해줘. - 일본어 발음을 한국어로 옮겨 적지 마.
